IEEE CS MEC SB is organizing a learner program to help students learn by building the SB website

February 20, 2019

The IEEE CS MEC SBC is happy to announce that we will be conducting a mentorship program in partnership with FOSSMEC to help first timers learn the nitty gritty of web development by building the SB website and the same shall be done in a FOSS manner on the SB GitHub Organisation where anyone can send their suggestion for this project can make the same in the repository issues section.
