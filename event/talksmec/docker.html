Talk on Docker

Kurian Benoy of CS6A presented on containerization with Docker. Docker is tool used in Infrastructure As A Service(IAAS) where various applications are serviced by single server entity. Kurian is an active contributor to FOSS organizations and has spoken at FOSS Asia, Singapore and PyCon India 2019, Chennai. He is contributing regularly towards building better data managment tools, such as the Data Version Control (DVC) project.
